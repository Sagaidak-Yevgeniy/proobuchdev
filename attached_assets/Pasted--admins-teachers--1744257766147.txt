Функционал для преподавателей (admins/teachers):

Преподаватель может создавать курсы с различными модулями:

Название курса

Описание

Программа (лекции и задания)

Баллы за каждый урок/задачу

Преподаватель может добавлять задания к лекциям или курсам:

Задания с условиями (с возможностью добавления тестов)

Проверка решений на основе тестов

Преподаватель может добавить баллы за каждое задание

Ограничения по времени выполнения задания (если необходимо)

Преподаватель может назначать минимальный проходной балл для каждого задания или курса

Функционал для студентов (students):

Студенты могут записываться на курсы и выполнять задания

Задания должны быть проверены и баллы начислены только после правильного выполнения

Студенту нужно получить минимум баллов на курсе, чтобы продолжить к следующему уроку или лекции

Студент может просматривать результат выполнения задания (пройдено/не пройдено)

Прогресс по курсу должен отображаться с количеством баллов

После завершения курса или олимпиады и достижения проходного балла, студенту выдается сертификат

Система тестирования и проверки решений:

Задания должны проверяться автоматически по тестам

Если студент правильно решает задание, начисляются баллы

Баллы за выполнение задач суммируются для курса или олимпиады

Система должна проверять ответы студентов и сравнивать с ожидаемым результатом

Преподаватель может в любой момент обновить условия задания, а также тесты

Сертификаты и достижения:

Система должна генерировать сертификат с уникальным кодом по завершению курса или олимпиады

Сертификат выдается студенту, если он набрал необходимое количество баллов

В сертификате указаны данные о курсе, студенте и результатах

Студенты могут скачать сертификат с персональной страницы

Прогресс и рейтинг:

Студентам начисляются баллы за каждое выполненное задание

Рейтинг студентов по курсу или олимпиаде (с возможностью фильтрации по баллам)

Панель администратора для просмотра статистики по курсам и олимпиадам

Адаптивный интерфейс:

Все страницы должны быть адаптивными (под смартфоны, планшеты, десктопы)

Используй современный подход к интерфейсу с кнопками, меню и панелями

Простой, но современный дизайн с использованием Tailwind CSS

Меню должно быть интуитивно понятным: страницы курсов, заданий, профиля, сертификатов

Пожелания по функционалу:

Преподаватели могут создавать разные виды заданий (например, программирование, тесты с выбором, теоретические вопросы)

Студенты должны пройти каждый урок, чтобы перейти к следующему

У каждого задания могут быть разные баллы в зависимости от сложности

Лекции должны быть разбиты на несколько этапов: лекция — задание — итоговая проверка

Технические требования:

Генерация моделей для Курсов (Course), Заданий (Task), Студентов (Student), Результатов (Result), Сертификатов (Certificate)

База данных PostgreSQL для хранения информации о пользователях, заданиях, результатах и сертификатах

Реализуй систему авторизации с ролями (студент, преподаватель, администратор)

Реализуй систему создания, редактирования и удаления курсов и заданий

Страница студентов с отображением прогресса по курсу, результатами и сертификатами

Для каждой олимпиады создаётся уникальный рейтинг участников
